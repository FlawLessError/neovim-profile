# Zebar Bar Gruvvy Watermelon

This is a Zebar bar theme based on [Gruvvy Watermelon](https://github.com/zacccharv/gruvvyWatermelon) and forked from [Zebar Rose Pine](https://github.com/thearturca/zebar-rose-pine-bar).

It is a simple bar with the Gruvvy Watermelon theme and glazewm workspaces support.

### Features:

- glazewm workspaces support
- media player support
- time support
- focused window title display
- cpu, memory usage, keyboard layout support
- weather support

## Usage

```bash
$ pnpm install
```

## Available Scripts

In the project directory, you can run:

### `pnpm run dev`

Runs the app in the development mode.

### `pnpm run build`

Builds the app for production to the `dist` folder.<br>
It bundles Solid in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.<br>
